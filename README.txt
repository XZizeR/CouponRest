Compact management system using Spring Boot.
Utilizing login system where you need to authorize your identity in the system,
through user-name and user-password. Enables you using coupons as  an object
connected to you, through the SQL data managing services by Hibernate and
Jakarta Persistence (JPA). The platform able to add, update, perform data by
logged user, with a thread that deletes out-dated coupons, and can connect to
Angular through Rest.
